## 用法

```
nhctl list [NAME] [flags]

Aliases:
  list, ls
```

## 标记

```
Flags:
      --full   list application meta full
  -h, --help   help for list
      --json   use json as out put, only supports for 'nhctl list'
      --yaml   use yaml as out put, only supports for 'nhctl list'

Global Flags:
      --debug               enable debug level log
      --kubeconfig string   the path of the kubeconfig file
  -n, --namespace string    kubernetes namespace
```
